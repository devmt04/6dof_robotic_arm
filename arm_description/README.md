# Robotic Arm Description

ROS 2 robot description package for a 6-DOF serial robotic manipulator.

This package contains the robot's URDF/Xacro model, visual and collision meshes, RViz configuration, and launch files required for visualization and simulation.

## Package Structure

```text
urdf/
meshes/
launch/
rviz/
config/
```

## Contents

* URDF/Xacro robot model
* Visual and collision meshes
* RViz configuration
* Robot visualization launch files
